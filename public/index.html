Fake Manufacturing Data API

This API provides endpoints for Users, Products, Customers, and Orders. Data is stored in-memory for demo purposes.

Base URL

https://fake-manufacturing-data-api.onrender.com

Endpoints

Users

  • GET /users - Returns a list of users
  • POST /users - Add a new user. Required fields: name

Products

  • GET /products - Returns a list of products
  • POST /products - Add a new product. Required fields: name, price, sku

Customers

  • GET /customers - Returns a list of customers
  • POST /customers - Add a new customer

Orders

  • GET /orders - Returns a list of orders
  • POST /orders - Add a new order. Required fields: customer_id, items

See the GitHub README for full details and examples.
